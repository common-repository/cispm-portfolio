== Installation ==
**Français**

Téléchargez *Cispm Portfoliot* sur votre blog : 

Pour ce faire, télécharger et dézipper le fichier *cispm_portfolio.zip*. 

Copiez le par FTP dans le dossier *wp-content/plugins/* du serveur qui héberge votre blog WordPress.

Pour activer le plugin, rendez-vous dans l'interface administration de WordPress, section *Extensions* et cliquez sur "activer".

Pour installer le plugin sur une page WordPress, ajouter dans votre page le code html [cispm-portfolio] **IMPORTANT : Choisir l'onglet HTML.**

Dans les *Réglages* de l'interface administration de WordPress une nouvelle section apparaît nomme "Cispm portfolio", c'est ici que vous pouvez personnaliser l'apparance de votre portfolio.

A tout moment, vous pouvez revenir au thème par défaut en cliquant sur le bouton correspondant.

Les couleurs des éléments de votre portfolio sont personnalisable en caractère hexadécimal selon les normes CSS (ex: #FF0000 pour rouge).

Les polices de caractères peuvent aussi se personnaliser.

**English**

Download *Cispm Portfolio* on your blog : 

To do this, download and unzip the file *cispm_portfolio.zip*.

Copy this into FTP folder *wp-content/plugins/* of server who hosts your WordPress blog.

To activate the plugin, go to the administration interface of WordPress, section *Extensions* and click "Enable".

To install the plugin on a WordPress page, add in your page html the code [cispm-portfolio] **IMPORTANT: Select the HTML tab.**

In the *Settings* interface administration of WordPress is a new section called "Cispm portfolio", this is where you can customize the appearance of your portfolio.

At any time you can return to the default theme by clicking the appropriate button. The colors of your portfolio elements are customizable character in hexadecimal as CSS standards (ex: # FF0000 for red).

The fonts can also be customized.
